                             Nix reference manual#

   The Nix reference manual is available for multiple versions:

     * Nix pre-release

       Development build from the master branch of the Nix repository

     * Nix 2.24 (single page)

       Latest Nix release

     * Nix 2.18 (single page)

       Shipped with the rolling release of Nixpkgs and NixOS

     * Nix 2.18 (single page)

       Shipped with the current stable release of Nixpkgs and NixOS: 24.05

     * Nix 2.18 (single page)

       Shipped with the previous stable release of Nixpkgs and NixOS: 23.11
