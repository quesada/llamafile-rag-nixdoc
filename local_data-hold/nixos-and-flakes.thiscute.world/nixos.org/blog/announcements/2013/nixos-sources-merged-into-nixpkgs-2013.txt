                       NixOS sources merged into Nixpkgs

   The NixOS Git tree has been merged into the Nixpkgs tree in order to
   simplify development. The sources now live in the nixos subdirectory of
   the Nixpkgs repository on GitHub. See the announcement for more
   information.
