      #13 - Cross compilation for Go and Rust, home-manager email support

      #13 - Cross compilation for Go and Rust, home-manager email support

                                      News

     * NixOS 19.03 will leverage multiple cores automatically by default for
       builds and Nix 2.2 will have the sandbox enabled by default on Linux

     * Deploy a C++ web app on Heroku using Docker and Nix

     * Meson 0.48 landed in staging again

     * Declarative client mail configuration

       In the past few months, some work went into home-manager to abstract
       mail configuration and generate configs for a variety of client-side
       software. Declare your username/password once and generate configs for
       afew/alot/astroid/mbsync/offlineimap.

     * Go cross-compilation landed in staging

       This includes support for Cgo and buildGoPackage. Rust
       cross-compilation could follow soon.

     * RFC on improving RFC process

     * Released cachix 0.1.3

     * Call for new stickers design for events

     * OfBorg now uses checks for build status reports, greatly reducing
       Nixpkgs reviewers’ email load.
