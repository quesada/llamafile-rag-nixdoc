Tutorials & Resources

     * fromElisp

       An Emacs Lisp reader in Nix.

     * dconf2nix

       Nixify your DConf files (Gnome3 configuration) as expected by Home
       Manager’s dconf settings.

     * pynixify - Nix expression generator for Python projects

       A tool made to adopt Nix in your Python project, and to contribute to
       Nixpkgs with package definitions.

     * Experimenting with Nix and the service management properties of Docker

       Demonstrates how we can combine Nix and Docker in unorthodox ways — we
       can replace traditional Docker images for dependency management by a
       shared Nix store, use Docker as a backend for my experimental Nix
       process management framework and use Disnix as a simple orchestrator
       for Docker containers.

     * Manix - Documentation Searcher for Nix

       A fast Documentation Searcher for Nix. Its database consists of
       Nixpkgs Documentation, Nixpkgs Comments, Nixpkgs Tree (pkgs.,
       pkgs.lib.), NixOS Options and Home-Manager Options.

     * Searching and installing old versions of Nix packages

       Installing older versions of packages in Nix is easy but currently
       there is no official way to find out what revision has the package
       version that I need.

     * Local Nix without Root

       Monkeying around with nix for HPC systems which have no root access
       and NFS filesystems.

     * BigBlueButton packaging and modules for NixOS

       First preliminary release of bbb4nix, a way to run BigBlueButton on
       NixOS.
