                                      News

     * Moving from Homebrew to Nix Package Manager

       Salar wrote a step-by-step tutorial how to migrate from homebrew to
       Nix.

     * nixos/printers: declarative configuration

       Tired of manually configuring your CUPS printers every time you set up
       a new NixOS host? Please take a look and give it a try to see whether
       everything works with your setup!

     * Static Nix: a command-line swiss army knife

       Matthew shows how using statically linked Nix in a 5MB binary, one can
       use Nix without root. With an one-liner shell, you can use Nix to
       install any software on a Linux machine.
