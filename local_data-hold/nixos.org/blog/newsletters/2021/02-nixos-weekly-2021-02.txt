                               #02 - NixOS Weekly

                                      News

Announcements

     * Introducing flox - Nix for simplicity and scale

       flox makes it easy to build software or manage development
       environments with Nix at any scale.

     * Trustix

       Distributed trust and reproducibility tracking for binary caches.

     * Recurring funding for making macOS a first-class citizen in Nix
       ecosystem

       A call for companies/developers to contribute financially towards
       addressing major macOS issues in the ecosystem.

Tutorials

     * Offloading NixOS builds to a faster machine

     * Build your NixOS Raspberry Pi Image using Github Action

     * Safe service upgrades using system.stateVersion

Resources

     * Developing an s6-rc backend for the Nix process management framework

     * envfs

       Fuse filesystem that returns symlinks to executables based on the PATH
       of the requesting process (fixes shebang problems on NixOS).

     * nixpkgs-review-checks

       Add additional checks and more information from build logs and outputs
       to the reports generated by nixpkgs-review.

     * Podcasts about Nix

       A content addressable store for Nix with Théophane Hufschmitt and
       Binary Verification with Trustix starring Adam Höse.
