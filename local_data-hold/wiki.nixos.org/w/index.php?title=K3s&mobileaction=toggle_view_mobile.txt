   From NixOS Wiki

   K3s is a simplified Kubernetes version that bundles Kubernetes cluster
   components into a few small binaries optimized for Edge and IoT devices.

   NixOS's K3s documentation is available at:

   https://github.com/NixOS/nixpkgs/blob/master/pkgs/applications/networking/cluster/k3s/README.md
