   Godot is an open-source game engine, capable of creating both 2D and 3D
   games.

Usage in NixOS

   Use steam-run to run a Godot exported project (without having to fix paths
   for Nix).

Installation

   Add pkgs.godot_4 to environment.systemPackages.
